Docker 101 at Esprit


This a Docker workshop organized by ESPRIT Club Cloud'Tic where I talked about some of the basics of Docker and containerization in general.
I have discussed some of the points below:

  • Namespaces, cgroups, rootfs
  • Docker 101 (how to use docker)
  • Dockerfiles
  • Brief explanation of the Moby project
  • Docker daemon, Docker CLI, Rest API, Docker engine and what the hell is the role of the Linux kernel
